# Clone-Xs UI

React frontend built with Vite, TypeScript, Tailwind CSS 4, and shadcn/ui.

## Development

```bash
npm run dev     # Start dev server (port 3000)
npm run build   # Build for production
npm run preview # Preview production build
npm run lint    # Run ESLint
```

Or from the project root:

```bash
make web-start  # Start API + UI together
make build-ui   # Build frontend for production
```
